CNT Metrics

This project aims to automate the diameter measurement for carbon nanotubes. A model developed using Python, TensorFlow (Keras), Pandas, and OpenCV to classify and measure individual CNTs within tunneling electron microscopy (TEM) images. A pipeline was developed to create training data from the provided manually measured CNT images. The model can currently distinguish regions of interest that contain a CNT suitable for measurement from regions of interest that do not.

SubRes

SubRes aims to help applicants maintain multiple versions of their resume and other application materials by allowing users to dynamically add and remove skills and experience from their resumes. First, users create a master resume with all possible skills, job experience bullet points, and summary. This master resume (CV, if you will) is saved to a postgreSQL database for later access. Then, the user can then downselect from this master resume to create a tailored resume simply by selecting which items to include or not. This subselection of items can be saved and then used to generate a tailored resume. The master resume and subselections can be updated later, as the user needs. By tying each tailored resume to a master resume, updating and generating tailored resumes becomes more controlled and less tedious.
